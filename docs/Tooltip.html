Tooltip

A tooltip is a small text container that is overlaid on top of other elements in a page to display additional information when triggered via mouse interaction.

Placement

A tooltip is usually used for displaying optional information that doesn't need to be always visible for everyone. It can also be used to contain more detailed versions of texts to reduce the amount of overall texts in a page to avoid overwhelming users. An element that triggers a tooltip is usually marked by an icon or a font styling that makes it stand out from the rest of the interface.

Behavior

When a mouse cursor moves on an element that triggers a tooltip, a tooltip is displayed. When a mouse cursor leaves the element, the tooltip is removed from the view.

Accessibility Guidelines

Careful consideration is needed regarding color schemes to make it easily distinguishable from the background. It should be also clear on which element this tooltip is providing extra information.

Instructions

  1. Assign a tooltip class name to an element that triggers a tooltip. The naming convention is 'tooltip-{arrow position name}-{color}'. Arrow positions consists of two words, and an arrow position name takes each of first letter such as 'tl' from 'top left', 'rb' from 'right bottom.'
    • Tooltip with an arrow at bottom left and a light background: 'tooltip-bl-light'
    • Tooltip with an arrow at left center and a dark background: 'tooltip-lc-dark'
  2. Create a div with a class tooltip-content which is the tooltip itself.
  3. Create the following elements inside for tooltip texts and append to tooltip-content above:
    • tooltip-text-title: [optional] title of texts in the tooltip
    • tooltip-text-body: main texts in the tooltip

Sample Code

<div class="tooltip-bc-light">This is a tooltip
   <div class="tooltip-content">
      <span class="tooltip-text-title">Title</span>
      <span class="tooltip-text-body">Tooltip message</span>
   </div>
</div>
