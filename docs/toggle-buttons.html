Toggle Button

Meant to activate a setting that has two states: on or off. Also to be used when flipping the toggle immediately performs an action on the page. Requires javascript to act on click.

Placement

Can be placed anywhere on a page. If you are going to have a group with two opr more toggles, place them in a fieldset.

Behavior

Clicking on the toggle changes its state from on to off, or vice-versa.

Accessibility Guidelines

This gets into UX as well as accessibility. There are two common ways to create a toggle: one using an input field of type checkbox with 'role=switch', and one using the button element with 'aria-pressed' attribute. There are varying opinions, but the general consensus leans toward using the button element for a toggle switch. DeQue recommends buttons, and Adrian Roselli explains it this way:

Use <button> if:

  • you can count on JavaScript being available, and
  • flipping the switch has an immediate effect.

Use <input type="checkbox"> if:

  • you want to progressively enhance the control, and
  • flipping the switch will only take effect when the user submits it.
